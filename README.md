# GVEDC-Learning-Coordinato-Agent

AI Agent 学习团队 - 知识图谱与向量数据库协调学习系统

## 团队成员

- **Learning Coordinator** - 学习协调Agent
- **Researcher Agent** - 研究Agent
- **Implementer Agent** - 实现Agent
- **Documenter Agent** - 文档Agent

## 核心功能

- 多Agent协作学习
- 项目深度分析与研究
- 论文与百科生成
- 知识图谱构建
- GVEDC数据库存储

## 技术栈

- Python 3
- SQLite (数据持久化)
- ChromaDB (向量检索)
- Git (版本控制)

## 使用方法

```bash
python run_agents.py
```

## 项目结构

```
GVEDC-Learning-Coordinato-Agent/
├── learning_coordinator.py   # 学习协调器
├── researcher_agent.py        # 研究Agent
├── implementer_agent.py       # 实现Agent
├── documenter_agent.py       # 文档Agent
├── unified_memory_system.py  # 统一记忆系统
├── run_agents.py             # Agent启动脚本
└── README.md
```
