## 3. 核心知识点详解

### 3.1 Python基础

#### 3.1.1 变量和数据类型

Python中的变量不需要声明类型，解释器会自动推断。

```python
# 基础类型
name = "Python"       # 字符串
age = 30              # 整数
price = 99.9          # 浮点数
is_active = True      # 布尔值

# 容器类型
numbers = [1, 2, 3]           # 列表
coordinates = (10, 20)          # 元组
person = {"name": "Tom"}        # 字典
unique_ids = {1, 2, 3}          # 集合
```

#### 3.1.2 控制流程

```python
# 条件判断
if age >= 18:
    print("成年")
elif age >= 6:
    print("青少年")
else:
    print("儿童")

# 循环
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

# 列表推导式
squares = [x**2 for x in range(10)]
```

#### 3.1.3 函数和模块

```python
# 函数定义
def greet(name, greeting="Hello"):
    """问候函数"""
    return f"{greeting}, {name}!"

# Lambda表达式
square = lambda x: x ** 2

# 模块导入
import os
from datetime import datetime
```

### 3.2 面向对象编程

```python
class Animal:
    """动物类"""
    def __init__(self, name):
        self.name = name

    def speak(self):
        raise NotImplementedError

class Dog(Animal):
    """狗类"""
    def speak(self):
        return f"{self.name} says Woof!"

# 多态
def make_speak(animals):
    for animal in animals:
        print(animal.speak())
```

### 3.3 并发编程

```python
import threading
import asyncio

# 多线程
def worker():
    print("Working")

thread = threading.Thread(target=worker)
thread.start()

# 异步编程
async def fetch_data():
    async with aiohttp.ClientSession() as session:
        async with session.get(url) as response:
            return await response.text()
```

### 3.4 Web开发

```python
# Django示例
from django.views import View
from django.shortcuts import render

class IndexView(View):
    def get(self, request):
        return render(request, 'index.html', {'message': 'Hello!'})
```

## 5. 学习建议

### 5.1 学习方法

1. **循序渐进**: 不要急于求成，每个知识点都要理解透彻
2. **动手实践**: 光看不动手是学不会编程的
3. **代码规范**: 从一开始就养成良好的编码习惯
4. **及时复习**: 定期回顾之前学过的内容
5. **项目驱动**: 通过项目来巩固和应用所学知识

### 5.2 学习时间安排

| 阶段 | 时长 | 每天学习时间 |
|------|------|-------------|
| 基础阶段 | 15天 | 2-3小时 |
| 进阶阶段 | 5天 | 3-4小时 |
| Web基础 | 10天 | 3-4小时 |
| Python高级 | 20天 | 4-5小时 |
| 框架开发 | 20天 | 4-5小时 |
